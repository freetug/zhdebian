ViEmu/SQL: vi-vim for
SQL Server

Vi-vim for your database work

Many developers and database administrators spend a long time working inside Microsoft's database workbench. ViEmu for SQL Server allows you to use vi-vim editing even inside this environment.

There are several versions of ViEmu for SQL Server: ViEmu/SQL Server "Classic" works with SQL Server 2005 and 2008 Management Studio, ViEmu/SQL Server 2012/14 works with SQL Server 2012 and SQL Server 2014, ViEmu/SQL 2016/2017 works with SQL Server 2016 & 2017 Management Studio versions, and ViEmu/SSMS18 works with the latest stand-alone SQL Management Studio 18 download.
