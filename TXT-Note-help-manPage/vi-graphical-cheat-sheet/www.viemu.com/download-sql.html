Free 30 day trial

ViEmu for SQL Server 2005 and 2008 Management Studio will work for 30 days after the date of installation. If you want to keep using it, you need to purchase a license.

Installation instructions

In order to install, double click on the EXE file once you have downloaded it. Have SQL Server Management Studio closed while you install. The installer will automatically update any installation of a previous version if there is one.

To start using it, once in Management Studio, just open any source or text file and start editing as if you were using vi or vim - ViEmu will make vi/vim-style input available in all text editing windows.
