Installation instructions

In order to install, double click on the VSIX file once you have downloaded it. The built-in VS installer will automatically update any installation of a previous version if there is one.

To start using it, once in Visual Studio, just open any source or text file and start editing as if you were using vi or vim - ViEmu will make vi/vim-style input available in all text editing windows.
