ViEmu/VS: vi-vim editing
for Microsoft Visual Studio

vi-vim for development productivity

ViEmu for Visual Studio is the most popular and mature product of the ViEmu line. Many hundreds of developers use it all day, every day for all their development tasks. Supporting all basic and many advanced features of vi/vim inside Visual Studio, it's a boost for your development productivity.

ViEmu for Visual Studio "Classic" is compatible with Visual Studio 2008, Visual Studio 2005, Visual Studio .NET 2003, and Visual Studio .NET 2002. There are three other builds, one for VS2010 and VS2012, another one just for VS2013, and finally another one for VS 2015, VS 2017 and the latest VS 2019 IDE.
